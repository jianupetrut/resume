Web Application Security
Fall 2019

In the first official security course, after learning the fundamentals about information security and the basics of web application penetration testing, we received a group assignment where we had to play as both blue and red team along with another group of classmates.
All teams started with a very vulnerable virtual machine which hosted a web server and we had to harden it, but leave one weak vulnerability, one backdoor to allow the red team to gain a remote shell and one way to gain root privileges from it - this represented the blue team assignment.
When playing the red team, we were supposed to hack the other group's webserver. We found a vulnerability in their file upload mechanism which we used to gain a remote shell as www-data and exploited a kernel vulnerability for privilege escalation.
I consider this project my top favorite one, not only because it was a great learning experience but also because I enjoyed it more than any other project I have ever worked on and it was the definite moment that made me realize that I want to work with cybersecurity for the rest of my life.

Quiz Application
